# 🛒 E-commerce Store Annual Sales Analysis (Power BI Dashboard)

## 📌 Project Overview
This project focuses on analyzing an e-commerce retail dataset using Power BI to uncover insights related to sales performance, customer behavior, and business trends. The goal is to support data-driven decision-making and improve overall business strategy.

---

## 🎯 Problem Statement
The e-commerce business lacked clear visibility into:
- Annual sales performance  
- Customer purchasing behavior  
- Product-wise revenue trends  
- Key performance metrics (KPIs)  

This made it difficult to track growth and optimize sales strategies.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Understand customer buying patterns  
- Identify top-performing products  
- Track key business KPIs  
- Build interactive dashboards for insights  

---

## 🗂️ Dataset Description
The dataset includes:
- Order details (Order ID, Date)  
- Customer information  
- Product categories  
- Sales and profit data  
- Quantity and pricing  

---

## 🛠️ Tools & Technologies
- **Power BI** – Dashboard creation & visualization  
- **Power Query** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – KPI calculations  

---

## 📊 Dashboard Features

### 🔹 Sales Overview
- Total sales and profit  
- Monthly and yearly sales trends  
- Revenue growth analysis  

---

### 🔹 Customer Insights
- Customer segmentation  
- Repeat vs new customers  
- Customer purchasing behavior  

---

### 🔹 Product Performance
- Top-selling products  
- Category-wise sales analysis  
- Profit contribution by products  

---

### 🔹 KPI Metrics
- Total Revenue  
- Total Orders  
- Total Profit  
- Average Order Value  

---

### 🔹 Interactive Features
- Slicers and filters for dynamic analysis  
- Drill-down capabilities  
- User-friendly dashboard design
